# House Price Prediction using Linear Regression from Scratch

## Overview
A Machine Learning project implementing Simple Linear Regression from scratch using Gradient Descent.

## Features
- Data preprocessing
- Feature normalization
- Cost function implementation
- Gradient Descent optimization
- Price prediction
- Data visualization

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Google Colab

## Dataset
House Price India Dataset

## Results
The model successfully learns the relationship between living area and house price and visualizes the regression line against actual data.

## Future Improvements
- Multiple Linear Regression
- Additional house features
- Train/Test Split
- Performance Metrics
